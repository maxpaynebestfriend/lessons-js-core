    Однажды меня очень сильно заинтересовал термин "программирование". Затем у меня возникла масса вопросов, на которые я очень жаждал получить ответы. В океане течений IT мира - одно из направлений увлекло меня сильнее остальных. Web-программирование! - первые разметки страниц на HTML, добавление веб-страницам стилей через css. А мое знакомство с JavaScript произошло после того, как в руки попала книга "Программирование для детей на JS". После ее изучения, а также после находки прекрасного сайта learn.javascript.ru - у меня сформировались навыки оживления страниц, используя JS, словно магические заклинания древности.
